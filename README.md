# Здесь должно быть имя игры...

### Требуется:
- Git LFS 
- Godot 4.6.2 (recommended)

### Установка Git LFS
Если Linux (apt):
```bash
sudo apt install git-lfs
git lfs install
```

Если Windows, то через [сайт](https://git-lfs.com/) устанавливаем

После того надо установить его:
```powershell
git lfs install
```
---
После каждого pull/fetch, если в проекте появился новый файл(от git lfs), то надо ранить:
```bash
git lfs pull
```
